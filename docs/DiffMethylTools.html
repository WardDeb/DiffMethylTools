merge_tables(case_data: InputProcessor, ctr_data: InputProcessor, min_cov=10, cov_percentile=100.0, min_samp_ctr=2, min_samp_case=2) → DataFrame[source]

Merge case and control data tables.

Note

case_data and ctr_data must be from a list of samples, where each contains one of the following column formats:

  • ["chromosome", "position_start", "coverage", "methylation_percentage"]

  • ["chromosome", "position_start", "positive_methylation_count", "negative_methylation_count"]

  • Additionally, "position_end" and "strand" columns can be used. These will be considered during the join process.

Parameters:

  • case_data (InputProcessor) – The case data to be merged.

  • ctr_data (InputProcessor) – The control data to be merged.

  • min_cov (int, optional) – Minimum coverage filter, defaults to 10

  • cov_percentile (float, optional) – Maximum coverage filter (percentile of sample coverage). Ranges from 0.0-100.0, defaults to 100.0

  • min_samp_ctr (int, optional) – Minimum samples in control, defaults to 2

  • min_samp_case (int, optional) – Minimum samples in case, defaults to 2

Returns:

Merged data table

Return type:

pd.DataFrame

window_based(data: ~lib.input_processor.InputProcessor | None = None, statistical_test: ~typing.Callable = <function mannwhitneyu>, window=1000, step=500, min_nbr_per_win=5, processes=12, min_std=0.1) → DataFrame[source]

Perform window-based analysis.

Note

data must contain the following column format:

  • ["chromosome", "position_start", "methylation_percentage*", "avg_case", "avg_ctr"]

  • There must be a methylation_percentage column for each sample to test.

Note

statistical_test must be a function from the scipy.stats package. Accepted functions include:

  • mannwhitneyu

  • ttest_ind

  • ranksums

  • ks_2samp

  • median_test

  • brunnermunzel

Parameters:

  • data (InputProcessor, optional) – Input data. Not necessary if the pipeline is in use, defaults to None

  • statistical_test (Callable, optional) –

    Statistical test to use, must be from scipy.stats package, defaults to mannwhitneyu

  • window (int, optional) – Sample window size (in bp), defaults to 1000

  • step (int, optional) – Window step size, defaults to 500

  • min_nbr_per_win (int, optional) – Minimum number of sample positions per window, defaults to 5

  • processes (int, optional) – Number of CPU processes, defaults to 12

  • min_std (float, optional) – Minimum standard deviation filter, defaults to 0.1

Returns:

Window-based analysis results

Perform position-based analysis. Has options for using the gamma function, or the limma R package.
